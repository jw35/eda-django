Reusing this data

This information is designed to be reused. All the pages above can be 'embedded' (by iframe) in other web sites (such as Google Sites ones) - see here for an example on the Association Web Site. The data can also be exported as Comma-Separated Values (CSV) which can be imported into spreadsheets for further analysis, and as GeoJSON which can be processed by Geographic Information Systems (GISs) and web mapping systems (this is how the maps above are created).

  • CSV exports: Towers, Contacts, Websites, Photos
  • GeoJSON: all towers, just Cambridge, just Ely, just Huntingdon, just Wisbech,
